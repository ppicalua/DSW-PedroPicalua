Proposito Del Sitio

Aprender HTML de manera fácil y sencilla en la implementación o esquema básico de una página web. HTML es el esqueleto o código base para el diseño de un página web, por lo que el propósito de esta página, Imágenesademas de aprender, es guiar el proceso de aprendizaje del estudiante durante todo el trayecto del curso para que pueda afianzar los conceptos básicos de HTML.

Etiquetas Basicas HTML

HTML es un lenguaje de marcas o etiquetas. Esto quiere decir que los elementos que conforman la página web se describen mediante palabras especiales que marcan el inicio y el final de los mismos, estas palabras se denominan etiquetas. Estas etiquetas se pueden agrupar según su función y son las que permiten la escritura HTML.

Atributos Etiquetas HTML

Al saber que HTML es un lenguaje de marcas o etiquetas, ha de saberse que esta etiquetas tienen unos atributos o propiedades que cambian la forma, el diseño o la función del contenido que se encuentra dentro de las etiquetas HTML. Estas propiedades o atributos son el color, el tamaño, la forma, la fuente, el tipo de letra, la posición, etc.

Valores De Etiquetas HTML

Establecer una dimensión, un tipo de letra, el tamaño de una imagen entre otras cosas en HTML implica asignarle unos valores a las propiedades o atributos de las etiquetas de marcado. Por ejemplo la propiedad width establece una dimensión explícita para la imagen. También hay una propiedad height correspondiente.

Etiquetas Layout En HTML

El diseño de páginas web conlleva una serie de requisitos o reglas para su construcción, por lo que se han establecidos una serie de normas o parámetros en el diseño de páginas, dichas normas son las etiquetas que se deben utilizar para cada una de las partes que conforman una pagina web, a esto se le conoce como construcción o diseño de layout.
